# Resource Matching Skill

Use this skill when the agent needs to match a user need to the support catalog.

## Matching Method

- Use only allowlisted catalog tools.
- Prefer resources whose tags match the detected need categories.
- Prefer campus or online resources when the request appears student related.
- Include crisis or emergency resources for immediate safety risk.
- Explain the top recommendation in plain language.
- Never invent real organizations, phone numbers, URLs, or eligibility rules.
